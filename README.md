# Math & Logic Utility Tool (JavaScript)

ブラウザ上で「数理計算」「論理判定」「文字列操作」を即座に実行できる軽量なWebユーティリティツールです。ユークリッドの互除法や素数判定アルゴリズムをJavaScriptで実装し、直感的なUIで提供します。

## 概要
本ツールは、日常的な計算やプログラミング学習における論理チェックをサポートするために開発されました。JavaScriptのモダンな構文（ES6以降）を活用し、エラーハンドリングとUXを考慮した設計になっています。

## 実装されている主な機能
*   **数理計算 (GCD / LCM)**:
    *   **最大公約数 (GCD)**: ユークリッドの互除法を用いた高速な計算アルゴリズム。
    *   **最小公倍数 (LCM)**: GCDを基にしたオーバーフローに配慮した算出。
*   **論理判定 (Prime / FizzBuzz)**:
    *   **素数判定 (Prime)**: $\sqrt{n}$ までの試し割り法による計算量の最適化（$O(\sqrt{n})$）。
    *   **FizzBuzz**: 剰余算を用いた古典的な条件分岐ロジックの実装。
*   **文字列比較 (String Compare)**:
    *   2つの文字列の完全一致判定（厳密等価演算子 `===` による評価）。

## 技術的ポイント
*   **DOM操作の最適化**:
    `document.getElementById` の結果が `null` の場合に備えた安全なイベント登録（`safeListen` 関数）を自作し、ランタイムエラーを防止しています。
*   **モダンな記述法**:
    分割代入 `[a, b] = [b, a % b]` による変数のスワップや、アロー関数による簡潔なロジック記述。
*   **セパレート構造**:
    HTML（構造）、CSS（装飾）、JavaScript（振る舞い）を分離し、保守性の高いコード構成を採用。

## 動作環境・セットアップ
### 推奨環境
*   Google Chrome / Microsoft Edge / Firefox（最新版）
*   **注意**: セキュリティ上の制限（CORS/Unique Origin）により、`file://` プロトコルでの直接実行ではなく、HTTPサーバー経由での閲覧を推奨します。

### 実行手順
1.  **VSCode Live Server** 等を起動し、`index.html` を開きます。
2.  または、ターミナルで以下のコマンドを実行します。
    ```bash
    python -m http.server 8000
    ```
3.  ブラウザで `http://localhost:8000` にアクセスしてください。

## ファイル構成
*   `index.html`: UI構造およびCSSスタイリング。
*   `hiko.js`: 計算ロジックおよびイベントリスナー制御。
